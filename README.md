Project Submission: Weather App 🌦️

I have developed a Weather Information Fetcher using Python and Streamlit. The application uses the OpenWeatherMap API to fetch real-time weather data for any city.

Features:
- Real-time weather updates
- Temperature, humidity, wind speed display
- Smart weather messages with emojis
- Clean and modern UI using Streamlit
